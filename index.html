üéØ Core Concepts

‚ö†Ô∏è Critical: This is NOT a Flat Form

  • This is a recursive tree structure where entities can own other entities
  • Reportability is computed based on effective ownership through chains
  • You must preserve the full chain to defend indirect ownership calculations
  • Cycle detection is mandatory to prevent circular references

What Problem This Solves

FinCEN regulations require drilling down through entity layers to identify reportable individuals:

  • 25%+ ownership (direct OR indirect)
  • Substantial control persons
  • Signing individuals

Example: Why Structure Matters

Why reportable: 100% of Mark LLC √ó 100% of Don LLC = 100% indirect ownership + substantial control role.

Data Structure

class PartyNode { id: string parent_id: string | null node_type: 'individual' | 'entity' | 'trust' role: 'owner' | 'trustee' | 'substantial_control' ownership_percentage: number | null children: PartyNode[] // ‚Üê RECURSIVE // COMPUTED (not stored): isReportable(): boolean calculateEffectiveOwnership(): number getDepth(): number }

Reportability Rules

An individual is reportable if ANY of these are true:

  • Role is "Substantial Control" ‚Üí ALWAYS reportable
  • Role is "Signing Individual" ‚Üí ALWAYS reportable
  • Effective ownership ‚â• 25% ‚Üí Reportable

Effective Ownership Calculation

Multiply ownership percentages up the parent chain:

// Example: Bob owns 60% of Holding Co, which owns 50% of Realty LLC Bob Johnson ‚îú‚îÄ Direct: 60% of Holding Co ‚îî‚îÄ Holding Co: 50% of Realty LLC Effective ownership = 0.60 √ó 0.50 = 0.30 (30%) ‚Üí ‚úì REPORTABLE (‚â• 25%)

üîß Implementation Guide

Critical Rules

1. Cycle Detection

Must prevent circular references at insertion time:

// BLOCKED: Creates circular reference Don LLC ‚Üí Mark LLC ‚Üí Mark Bowen ‚Üí Don LLC ‚ùå // Prevention logic: function hasAncestor(nodeId) { let current = this while (current.parent_id) { if (current.parent_id === nodeId) return true current = store.getNode(current.parent_id) } return false }

2. Deletion Rules

Cannot delete node if it has children:

function deleteNode(nodeId) { const node = store.getNode(nodeId) if (node.children.length > 0) { throw new Error( `Cannot delete: has ${node.children.length} children` ) } // OK to delete store.nodes.delete(nodeId) }

3. Computed Reportability

Never store as a boolean. Always calculate at runtime:

function isReportable() { if (this.node_type !== 'individual') return false // Always reportable roles if (this.role === 'signing_individual' || this.role === 'substantial_control') { return true } // Check ownership threshold return this.calculateEffectiveOwnership() >= 25 }

Database Implementation

Option 1: Relational (PostgreSQL)

CREATE TABLE party_nodes ( id VARCHAR PRIMARY KEY, parent_id VARCHAR REFERENCES party_nodes(id), transaction_id VARCHAR NOT NULL, node_type VARCHAR NOT NULL, role VARCHAR NOT NULL, ownership_percentage DECIMAL(5,2), data JSONB, certification_status VARCHAR DEFAULT 'pending' ); -- Traverse with recursive CTE: WITH RECURSIVE ownership_tree AS ( SELECT * FROM party_nodes WHERE parent_id IS NULL UNION ALL SELECT p.* FROM party_nodes p INNER JOIN ownership_tree t ON p.parent_id = t.id ) SELECT * FROM ownership_tree;

Key Operations

Add Child Node

function addChild(parentId, childData) { const parent = store.getNode(parentId) const child = new PartyNode(childData) // 1. Cycle detection if (parent.hasAncestor(child.id, store)) { throw new Error('Cycle detected') } // 2. Set parent relationship child.parent_id = parent.id // 3. Add to parent's children parent.children.push(child) // 4. Store in database store.addNode(child) return child }

Calculate Effective Ownership

function calculateEffectiveOwnership() { if (!this.ownership_percentage) return 0 let effective = this.ownership_percentage let current = this // Walk up parent chain while (current.parent_id) { current = store.getNode(current.parent_id) if (!current?.ownership_percentage) break // Multiply percentages effective *= (current.ownership_percentage / 100) } return effective }

Clone Tree for Repeat Buyer

function cloneTree(sourceId, targetId) { const sourceRoots = store.getRootNodes(sourceId, 'buyer') function cloneRecursive(sourceNode) { const cloned = new PartyNode({ id: `${targetId}_${sourceNode.id}`, transaction_id: targetId, ...sourceNode.data, created_from: sourceNode.id, certification_status: 'pending' // Force re-cert }) for (const child of sourceNode.children) { const clonedChild = cloneRecursive(child) cloned.addChild(clonedChild) } return cloned } return sourceRoots.map(cloneRecursive) }
